# GreenBytes-AI
## GreenBytes AI

GreenBytes AI is an intelligent storage optimization tool built for the IBM SkillsBuild & AICTE Virtual Internship. It scans designated storage folders to safely identify old, redundant, and temporary log files that contribute to data center energy waste. By locating and suggesting the removal of this digital clutter, it helps optimize cloud infrastructure and directly supports UN Sustainable Development Goal 13 (Climate Action) by lowering operational carbon footprints.
